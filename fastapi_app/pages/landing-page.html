Test Version! (Last updated on 30.03.2023)

PeopleSun is a planning tool for offgrid systems in Nigeria. The tool is used to plan distribution grids, to design energy supply by using solar modules, diesel engines, and battery storage, and to perform operational optimization. The tool was developed with the support of XYZ and has no commercial purpose, there are no fees for the users.

You can test the application without an user account. However, users are recommended to create an account to get access to larger computing capacity and to be able to store models.

Using the planing tool without registration

To test the planning tool, you do not need to register an account, a temporary account will be created automatically. However, if you use the planning tool without registering, please note:

• Your data will be automatically deleted after three hours

• If you log out, you lose access to your data
